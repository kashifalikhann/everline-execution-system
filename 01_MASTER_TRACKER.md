## Morris County P1 | Execution System v1.0

## HOW THIS SYSTEM WORKS

Every entity is assigned to ONE lane. Each lane has a specific action sequence.
No entity gets called without first checking if it can be done via portal or email.

### LANE DEFINITIONS:

**LANE A — PORTAL SELF-SERVICE (16 entities)**
- Entity has an online vendor registration form or bid portal
- ACTION: Fill out the form. Register Everline. Done.
- NO CALL NEEDED.
- Expected conversion: 100% (if form is filled correctly)

**LANE B — EMAIL DIRECT (40 entities)**
- Entity has a verified BA/QPA email
- ACTION: Send vendor package email. Follow up in 3 days if no reply.
- CALL ONLY if no reply after 2nd follow-up.
- Expected conversion: 25-40% response rate

**LANE C — CALL + EMAIL (23 entities)**
- Entity has no portal, no verified email, but has a real QPA/Clerk name
- ACTION: Call the named contact. Send email same day.
- Expected conversion: 40-50% when reaching the right person
